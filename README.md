# Bus Reservation System

A Java-based web application for bus ticket reservation and management.

## Features

- User registration and authentication
- Bus search and booking
- Payment processing
- Booking history
- Admin panel for bus and route management
- Support and request system
- Tariff management

## Technology Stack

- Java Servlets
- JSP (JavaServer Pages)
- HTML/CSS
- Eclipse IDE
- Apache Tomcat

## Project Structure

```
ticket/
├── src/main/
│   ├── java/          # Java servlet files
│   └── webapp/        # Web resources (CSS, images, JSP)
├── build/             # Compiled classes
└── .settings/         # Eclipse settings
```

## Setup

1. Import the project into Eclipse IDE
2. Configure Apache Tomcat server
3. Deploy and run the application

## Modules

- **User Module**: Registration, login, search, booking
- **Admin Module**: Bus management, route management, tariff updates
- **Support Module**: Customer support and request handling
- **Payment Module**: Payment processing for bookings
